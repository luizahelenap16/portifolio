~ Ferramentas Utilizadas ~

A linguagem usada para realização do Projeto de Desenvolvimento de Web Site, trata-se do HTML (Linguagem de Marcação de Hipertexto) é o código usado para estruturar uma página web e o conteúdo. Essa linguagem foi utilizada para o "corpo" das páginas, e funções básicas.

Para desing de texto, foi utilizado o CSS (Folha de Estilo em Cascata) é o código usado para dar estilo à página Web. Essa linguagem foi utilizada para o "desing" das páginas e edição dos textos.

Essas duas ferramentas, foram executadas à partir do VS Code Studio editor de código aberto.
